# Interface: Rgb

Object representation of sRGB color. Red, green, and blue values in the range
[0,255] and alpha values in the range [0,1].

**`Example`**

```
const rgb: Rgb = {
  r: 51,
  g: 102,
  b: 153,
  a: 0.8,
}
```
